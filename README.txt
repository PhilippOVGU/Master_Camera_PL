Executabe is in AllCode-->Opengl_Camera_Finder-->Debug
Anwendungshinweis:
ausführung in Commandline: Opengl_Camera_Finder.exe FileName.vtk mesh/lines
-->mesh wenn es sich um ein mesh handelt z.B. Aorta_mesh.vtk
--> lines wenn es sich um pathlines handelt z.B. Aorta_pathlines.vtk

WICHTIG: Aorta_mesh.vtk und Aorta_pathlines.vtk sind nicht enthalten wegen größe die müssen in den debug ordner gepackt werden

Beispielanwendung:
CMD.exe öffnen
cd ...\All_Code\Opengl_Camera_Finder\Debug
Opengl_Camera_Finder.exe prisma.vtk mesh

--> ein Prisma sollte auftauchen
Hinweis: prisma.vtk ist im upload enthalten
